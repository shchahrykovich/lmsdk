## Updates
1. git remote add upstream https://github.com/shchahrykovich/lmsdk.git
2. git fetch upstream
3. git checkout main
4. git merge upstream/main --allow-unrelated-histories -X ours --no-edit
5. git commit -m "Merge upstream/main (accept ours changes)"
6. git push origin main --force-with-lease
